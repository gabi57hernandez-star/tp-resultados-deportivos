## Escenario elegido
Escenario D - Estadísticas de Resultados Deportivos.
Análisis de resultados de partidos para generar estadísticas básicas del torneo.

## Dataset
- **Archivo:** datos/dataset.csv
- **Descripción:** Registro de 20 partidos de un campeonato con 6 equipos
- **Columnas:** fecha, equipo_local, equipo_visitante, goles_local, goles_visitante
- **Fuente:** Dataset simulado generado para el TP
